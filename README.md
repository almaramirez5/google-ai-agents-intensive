# Google AI Agents Intensive: Learning & Practice

This repository contains my personal notes and hands-on code from the **5-Day AI Agents Intensive Course by Google** (via Kaggle). 

As a Computer Engineering student (graduating in 2026), I'm using this course to level up my skills in AI agent architectures and get practical experience with Google's AI ecosystem (Gemini, Vertex AI, and ADK). 

## 🛠️ Tools & Tech I'm Learning
* **Models:** Google Gemini 
* **Frameworks:** Agent Development Kit (ADK), Model Context Protocol (MCP)
* **Deployment:** Google Cloud Platform (GCP) & Vertex AI
* **Language:** Python

## 📅 My Course Progress

### Day 1: Introduction to Agents & Multi-Agent Systems
* **What I learned:** The core differences between standard LLM applications and autonomous agent architectures.
* **What I built:** My first AI agent using Gemini and ADK with Google Search integration, plus a basic multi-agent system pattern.

### Day 2: Agent Tools & Interoperability (MCP)
* **What I learned:** How to give agents the ability to take actions using external APIs and the Model Context Protocol.
* **What I built:** Custom Python tools for the agent and a workflow with Human-in-the-Loop (HITL) approvals.

### Day 3: Context & Memory
* **What I learned:** Managing state and conversation history.
* **What I built:** Implemented short-term (Session) and long-term memory so my agents can maintain context across complex, multi-turn tasks.

### Day 4: Agent Quality & Debugging
* **What I learned:** How to evaluate, test, and debug AI agents.
* **What I built:** Set up basic observability (logs, traces, and metrics) to understand the agent's decision-making process.

### Day 5: Prototype to Production (Vertex AI)
* **What I learned:** Multi-agent communication and deployment basics.
* **What I built:** Explored the Agent2Agent (A2A) Protocol and deployed a test agent to **Vertex AI Agent Engine**.
